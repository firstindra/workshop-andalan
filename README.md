## 🚀 Fitur

- ✅ **Responsive Design** - Mobile & Desktop friendly
- ✅ **React Components** - Modular dan reusable
- ✅ **Smooth Scroll Navigation** - Navigasi halus antar section
- ✅ **Active Nav Highlight** - Menu aktif berdasarkan posisi scroll
- ✅ **Marquee Ticker** - Animasi teks berjalan
- ✅ **Dynamic Blog Cards** - Data blog dari array
- ✅ **Contact Form** - Form dengan state management
- ✅ **Like Button** - Fitur like/unlike pada blog posts
- ✅ **SVG Illustrations** - Ilustrasi kustom tanpa gambar eksternal
- ✅ **Pure CSS** - Tanpa library CSS tambahan

## 📦 Instalasi

### Prasyarat

- **Node.js** (versi 14 atau lebih baru)
- **npm** (biasanya sudah include dengan Node.js)

### Langkah-langkah

```bash
# 1. Clone repository
git clone https://github.com/USERNAME/workshop-andalan.git

# 2. Masuk ke folder project
cd workshop-andalan

# 3. Install dependencies
npm install

# 4. Jalankan development server
npm start
